customer/list

애플리케이션의 기능에 따라 url 경로와 view 파일이 저장되는 폴더를 추가 할 수 있습니다. 여기서는 customer 를 경로에 추가하고 view 파일도 customer 폴더에 저장했습니다.

/customer/list 는 Model 인터페이스를 이용해서 데이터(애트리뷰트)를 View customer폴더 list.html 에 전달하는 예시입니다. ⭕ 서버사이드렌더링에 타임리프 문법을 사용합니다.

애트리뷰트 출력 연습

  • 애트리뷰트 이름 page로 저장된 데이터(문자열) 출력
  • 애트리뷰트 이름 dto로 저장된 객체의 name 필드 출력
  • 애트리뷰트 이름 dto로 저장된 객체의 address 필드 출력
  • 애트리뷰트 이름 list 로 저장된 list 객체

CustomerMapper(dao) 에서 테이블의 데이터를 모두 조회한 결과를 보러 가기 여기 클릭!!
